# Bodhi Memorial Shrine 🐾

This is the interactive tribute site for Bodhi.

## Features
- Falling hearts & roses 🌹💖
- Candles with names 🕯️
- Drag & drop gifts 🎁
- Guestbook 📖 (messages persist)
- Balloons 🎈, Confetti 🎊, Lanterns 🏮
- Fireworks 🎇
- Theme toggle 🌗 (Serene/Emo)
- Music autoplay 🎶 (YouTube embed)
- Visitor counter and shareable link
- Mobile-friendly layout

## How to Deploy on GitHub Pages
1. Create a public repo on GitHub (e.g., `bodhi-shrine`).
2. Upload all files from this folder (`index.html`, `style.css`, `script.js`, `assets/`).
3. Go to **Settings → Pages** in the repo.
4. Under "Source", select branch = `main` and root folder.
5. Save, wait 1-2 minutes.
6. Your site will be live at:

   ```
   https://<your-username>.github.io/bodhi-shrine/
   ```
